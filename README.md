# Node.js Random Color Generator

When the program is run, print a 31x9 # block
in random color in hex code

```
node index.js
##############################
##############################
##############################
####                      ####
####       #ff0000        ####
####                      ####
##############################
##############################
##############################

```

- it should also accept hue color (red/green/blue)
- it should accept luminosity (light/dark)
